GEMGEM README
=============

Gemgem is a classic logic puzzle powered with the splashouille framework. The code is C++

DEPENDENCIES
============

To buil gemgem, you need the 1.2 Development Libraries for SDL, SLD_image and SDL_mixer.
You also need the libconfig version 9.0.
You need to compile and install splashouille first.

BUILDING GEMGEM
===============

Once you compile and install splashouille, go to the gemgem directory and run the command :

       make

RUNNING GEMGEM
==============

In the gemgem/ folder, just run ./gemgem and enjoy !
